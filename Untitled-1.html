What is HTML?

Definition

HTML also known or stands for HyperText Markup Language. It is a markup language used to create documents and designed to be displayed
in a web browser.

What have we learned in HTML?

HTML, despite it being used to create documents, it is also used to make a website and has flexible tools that is can use such as
css, java, javascript, html itself, and so on, and so forth. But, to require or attain these tools, we are to download and apply these
extentions before coding them and create your website accompanied with designs. In addition, using vscodium or vscode to create the html file, the UI is simple to understand and access.
